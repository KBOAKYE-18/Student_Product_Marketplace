# Student_Product_Marketplace

A Restful API that allows students to sellers to register , uplaod products and manage orders and reviews.  




## ‚ú® Features

- **User Authentication**
  - Register, Login, Logout
  - Password hashing using bcrypt
  - Protected routes using JWT

- **Product Management**
  - Students can create product listings
  - Edit and delete their own listings
  - Upload product images

- **Product Browsing**
  - View all available products
  - Search and filter functionality

- **Order System**
  - Place orders for products
  - View order history on your dashboard

- **Review System**
  - Buyers can review sellers and products
